# 🎮 游戏控制键

## 驾驶控制

| 按键 | 功能 |
|------|------|
| W | 前进 / 加速 |
| S | 后退 / 刹车 |
| A | 左转 |
| D | 右转 |

## 巴士功能

| 按键 | 功能 |
|------|------|
| O | 开门 (Open) |
| C | 关门 (Close) |
| Y | 上车 / 下车 |

## Roblox Studio 快捷键

| 快捷键 | 功能 |
|--------|------|
| F5 | 运行测试 |
| F8 | 停止测试 |
| F | 聚焦选中物体 |
| Ctrl+D | 复制 |
| Ctrl+Z | 撤销 |
| Ctrl+S | 保存 |
| Ctrl+G | 打组 |
| Alt+拖动 | 复制并移动 |

## 视角控制

| 操作 | 功能 |
|------|------|
| 右键+WASD | 移动视角 |
| 滚轮 | 缩放 |
| 中键拖动 | 旋转视角 |
