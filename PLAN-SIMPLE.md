# Day 1：认识 Roblox Studio

**今天目标：** 学会基本操作，搭一个小房子

---

## 第一步：打开 Studio 创建项目（10分钟）

1. 双击桌面的 **Roblox Studio** 图标
2. 等它打开（可能需要登录）
3. 点击 **New**（新建）
4. 点击 **Baseplate**（有地面的空白项目）
5. 等待加载完成

**保存项目：**
1. 点击左上角 **File**
2. 点击 **Save to Roblox As...**
3. 在 Name 框里输入：`Bus65`
4. 点击 **Create**

✅ 做完了？继续下一步！

---

## 第二步：认识界面（10分钟）

**找到这 4 个区域：**

```
┌─────────────────────────────────────────┐
│            菜单栏（顶部）               │
├─────────┬───────────────────┬───────────┤
│         │                   │           │
│Explorer │    3D 视口        │Properties │
│ (左边)  │   （中间大区域）  │  (右边)   │
│         │                   │           │
├─────────┴───────────────────┴───────────┤
│              Output（底部）             │
└─────────────────────────────────────────┘
```

**用鼠标点一下每个区域，确认你知道它在哪：**

- [ ] Explorer 在左边（显示游戏里所有东西的列表）
- [ ] Properties 在右边（显示选中东西的属性）
- [ ] 3D 视口在中间（你能看到游戏世界）
- [ ] Output 在底部（如果看不到，点 View → Output）

# Day 3：让巴士能开

**今天目标：** 写代码让巴士能用 WASD 控制

---

## 第一步：打开项目（2分钟）

打开 Roblox Studio，打开 Bus65 项目。

---

## 第二步：什么是脚本？（5分钟读一读）

**脚本就是给电脑的指令。**

比如我们要告诉电脑：
- 如果玩家按 W 键 → 巴士往前开
- 如果玩家按 S 键 → 巴士往后退
- 如果玩家按 A 键 → 巴士往左转
- 如果玩家按 D 键 → 巴士往右转

这些指令写成代码，电脑就能理解！

---

## 第三步：创建脚本（5分钟）

1. 在 Explorer 里找到 **VehicleSeat**（驾驶座）
2. **右键点击**它
3. 点击 **Insert Object**
4. 搜索 `Script`
5. 点击 **Script**

**一个新的 Script 出现在 VehicleSeat 下面了！**

---

## 第四步：打开脚本编辑器（2分钟）

1. **双击** Script
2. 打开了代码编辑器
3. 你会看到一行默认代码：`print("Hello world!")`

---

## 第五步：写你的第一行代码！（5分钟）

**删除那行默认代码，写上：**

```lua
print("巴士脚本启动了！")
```

**测试一下：**
1. 按 **F5** 运行
2. 看底部的 **Output** 面板
3. 应该显示：`巴士脚本启动了！`

🎉 你写的第一行代码生效了！

4. 按 **F8** 停止

---

## 第六步：准备工作（10分钟）

**在加驱动代码之前，要确保巴士零件不是"锁住"的：**

1. 在 Explorer 里展开巴士 Model
2. 按住 **Ctrl**，点击巴士里的**每一个零件**（Part、MeshPart 等）
3. 或者点击巴士 Model，然后按 **Ctrl+A** 全选
4. 看右边 Properties
5. 找到 **Anchored**
6. 确保它是**没有打勾**的 ❌

如果 Anchored 打勾了，巴士就锁在原地不能动！

---

## 第七步：复制驱动代码（15分钟）

**双击 Script 打开编辑器。**

**删除所有内容，然后复制粘贴下面的代码：**

```lua
-- 巴士驾驶脚本

local seat = script.Parent
local bus = seat:FindFirstAncestorOfClass("Model")
local body = bus.PrimaryPart

if body == nil then
    print("错误：请设置 PrimaryPart！")
    return
end

print("巴士准备好了！用 WASD 开车")

-- 速度设置（可以改这个数字！）
local speed = 40
local turnSpeed = 2

-- 创建让巴士动的力
local mover = Instance.new("BodyVelocity")
mover.MaxForce = Vector3.new(0, 0, 0)
mover.Parent = body

local turner = Instance.new("BodyGyro")
turner.MaxTorque = Vector3.new(0, 0, 0)
turner.P = 5000
turner.Parent = body

local angle = 0

-- 每一帧都检查
game:GetService("RunService").Heartbeat:Connect(function()
    if seat.Occupant then
        -- 有人坐在驾驶座上
        local forward = seat.Throttle  -- W/S 键
        local turn = seat.Steer        -- A/D 键
        
        -- 前进后退
        mover.MaxForce = Vector3.new(30000, 0, 30000)
        mover.Velocity = body.CFrame.LookVector * forward * speed
        
        -- 转向
        if math.abs(forward) > 0.1 then
            angle = angle - turn * turnSpeed
            turner.MaxTorque = Vector3.new(0, 50000, 0)
            turner.CFrame = CFrame.Angles(0, math.rad(angle), 0)
        end
    else
        -- 没人开车
        mover.MaxForce = Vector3.new(0, 0, 0)
    end
end)
```

---

## 第八步：测试！（10分钟）

1. 按 **F5** 运行
2. 走到巴士旁边
3. 按 **Y** 上车
4. 试试：
   - **W** = 前进
   - **S** = 后退
   - **A** = 左转
   - **D** = 右转

**能动吗？** 🚌💨

### 如果不能动：
- 检查所有零件的 Anchored 是不是取消勾选了
- 检查 PrimaryPart 有没有设置

### 如果巴士散架了：
别担心！明天我们会修复。今天先让它能动就行。

5. 按 **F8** 停止
6. **Ctrl+S 保存！**

---

## 第九步：调整速度（5分钟）

如果巴士太快或太慢：

在代码里找到这两行：
```lua
local speed = 40
local turnSpeed = 2
```

- `speed = 40` 改成 `speed = 30` 会变慢
- `speed = 40` 改成 `speed = 50` 会变快
- `turnSpeed = 2` 改成 `turnSpeed = 1` 转弯变慢

**改完后再 F5 测试！**

---

## ✅ Day 3 完成！

**今天你学会了：**
- [x] 创建 Script
- [x] 写第一行代码
- [x] 复制驱动代码
- [x] 巴士能开了！！

## 第二步：添加检测脚本（15分钟）

1. 右键 StopZone → Insert Object → Script
2. 双击打开
3. 删除默认代码，粘贴：

```lua
local zone = script.Parent

print("Richmond 站检测已启动")

local detected = false
```

2. `CloseDoor`

## 第三步：门控制脚本（15分钟）

在巴士 Model 里创建 Script，粘贴：

```lua
local bus = script.Parent
local door = bus:FindFirstChild("FrontDoor")

if door == nil then
    print("找不到门！")
    return
end

print("车门系统就绪 - O开门 C关门")

local ReplicatedStorage = game:GetService("ReplicatedStorage")
local openEvent = ReplicatedStorage:WaitForChild("OpenDoor")
local closeEvent = ReplicatedStorage:WaitForChild("CloseDoor")

local isOpen = false
local closedPosition = door.Position

openEvent.OnServerEvent:Connect(function()
    if isOpen then return end
    isOpen = true
    print("开门")
    door.Position = closedPosition + Vector3.new(2, 0, 0)
end)

closeEvent.OnServerEvent:Connect(function()
    if not isOpen then return end
    isOpen = false
    print("关门")
    door.Position = closedPosition
end)
```

---

## 第四步：键盘控制脚本（10分钟）

在 StarterPlayerScripts 里创建 LocalScript，粘贴：

```lua
local UserInputService = game:GetService("UserInputService")
local ReplicatedStorage = game:GetService("ReplicatedStorage")

local openEvent = ReplicatedStorage:WaitForChild("OpenDoor")
local closeEvent = ReplicatedStorage:WaitForChild("CloseDoor")

UserInputService.InputBegan:Connect(function(input, busy)
    if busy then return end
    
    if input.KeyCode == Enum.KeyCode.O then
        openEvent:FireServer()
    elseif input.KeyCode == Enum.KeyCode.C then
        closeEvent:FireServer()
    end
end)

print("按 O 开门，按 C 关门")
```
